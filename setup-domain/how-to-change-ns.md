# Как сменить NS

## Коротко

NS вводятся через пробел в одном сообщении, например: `ns1.example.com ns2.example.com`.

## Пошагово

1. Откройте домен.
2. Нажмите **Замена NS-записей**.
3. Посмотрите текущие NS.
4. Отправьте новые NS одной строкой через пробел.

## Что важно

- Можно использовать до 20 NS.
- Все NS должны быть рабочими.
- Новый список не должен совпадать с текущим.
- Не оставляйте старые NS в новом наборе.
