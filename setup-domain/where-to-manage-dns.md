# Где сейчас менять DNS

## Коротко

DNS меняются там, **чьи NS сейчас обслуживают домен**.

## Быстрая логика

- Если домен на NS Cloudflare — меняйте записи в Cloudflare.
- Если домен на 1REG / дефолтных NS — открывайте **1REG DNS** в боте.
- Если вы указали NS другого сервиса — меняйте записи у этого сервиса.

## Отдельно про MX

Если вам нужны почтовые записи, добавляйте их у того DNS-провайдера, который сейчас authoritative для домена.
