# Почта, язык и профиль

## Коротко

В профиле 1REG собраны основные вещи по аккаунту: ID пользователя, статус Prime, баланс, почта, язык, API и другие сервисные настройки.

## Почта

Привязанная почта нужна для уведомлений по доменам и получения кодов переноса.

## Язык

В боте можно переключаться между языками интерфейса.

## Когда это особенно важно

Перед запуском кампании лучше быстро пройтись по профилю и проверить базовые вещи заранее.
