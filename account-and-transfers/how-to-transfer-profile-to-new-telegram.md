# Как перенести профиль на новый Telegram

## Коротко

Если старый Telegram-аккаунт больше недоступен, профиль можно переносить только через ручную проверку.

## Что подготовить

- old Telegram ID
- new Telegram ID
- список доменов
- любые данные, которые подтверждают владение

## Что делать

Соберите данные заранее и напишите в поддержку одним сообщением.
