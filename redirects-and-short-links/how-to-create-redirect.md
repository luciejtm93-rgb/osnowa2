# Как сделать редирект

## Коротко

Редирект создаётся в карточке домена через кнопку **Редирект**. Нужно указать конечную статическую ссылку.

## Пошагово

1. Откройте домен.
2. Нажмите **Редирект**.
3. Вставьте конечную ссылку.
4. Сохраните.

## Когда редирект особенно удобен

Если домен нужен как промежуточный вход для кампании, чистый короткий переход или отдельная точка входа на лендинг.
