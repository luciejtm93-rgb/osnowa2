# Что значит 0/2

## Коротко

`0/2` — это checker по двум тревожным направлениям: Cloudflare и Google Safe Browsing. Если видно `0/2`, активных проблем по этим двум проверкам сейчас нет.

## Как читать статус

- **0/2** — активных срабатываний по проверкам нет
- **1/2** — есть одна проблема
- **2/2** — есть две проблемы одновременно

## Что важно понимать

`0/2` — это не гарантия, что с доменом вообще нет проблем. Это checker по двум конкретным типам рисков.
